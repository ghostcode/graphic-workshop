# 图文工坊

一个功能强大、界面精致的在线图片配字工具。无需后端服务，纯前端实现，所有数据处理均在浏览器本地完成，保护用户隐私。

## 功能特性

### 图片来源
- **颜色生成**：支持纯色背景生成，内置取色器与十六进制输入。
- **渐变背景**：支持线性渐变（左右、上下、对角）及径向渐变，可自定义起止颜色。
- **图片上传**：支持点击上传或拖拽上传本地图片，自动适配画布尺寸。

### 文字编辑
- **多行文本**：支持输入多行文字，自动换行显示。
- **丰富字体**：内置 7 种精选中文艺术字体（思源黑体、思源宋体、站酷快乐体、站酷庆科黄油体、站酷小薇体、马善政楷体、龙藏体）。
- **样式调节**：
  - 字号调节（12px - 200px）
  - 字重调节（100 - 900）
  - 文字颜色自定义（含预设快选）
- **文字特效**：
  - 描边效果：可调宽度和颜色。
  - 阴影效果：可调模糊半径和颜色。

### 交互体验
- **实时预览**：所有修改即时呈现在画布上，所见即所得。
- **拖拽定位**：支持在画布上直接拖拽文字调整位置。
- **预设位置**：提供九宫格快速定位按钮。
- **画布尺寸**：内置多种常用比例（4:3, 1:1, 3:4, 16:9, 9:16, 社交封面）。

### 导出
- 一键导出 PNG 格式图片。

## 技术栈

- **HTML5 Canvas**：用于图像绘制与合成。
- **Tailwind CSS**：用于快速构建现代化 UI 界面。
- **Vanilla JavaScript**：原生 JS 实现状态管理与交互逻辑。
- **Google Fonts**：加载高质量中文 Web 字体。

## 本地运行

本项目为单页面应用，无需构建工具。

1.  将代码保存为 `index.html`。
2.  在浏览器中直接打开该文件即可使用。

**注意**：由于使用了 Google Fonts 和 CDN 资源，运行时需要网络连接。

## 目录结构

## 使用指南

1.  **选择图片来源**：
    *   点击「颜色生成」使用纯色或渐变背景。
    *   点击「上传图片」导入本地图片。
2.  **编辑文字**：
    *   在左侧面板输入文字内容。
    *   选择字体、调整大小和颜色。
    *   如需特效，调整描边或阴影参数。
3.  **调整位置**：
    *   点击九宫格按钮快速对齐。
    *   或在右侧画布上按住文字直接拖动。
4.  **导出图片**：
    *   点击底部「下载图片」按钮保存成果。
